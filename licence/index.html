The Mute Bow is a Progressive Web App (PWA) created by Nicolas Gouwy and is available on Google Play as a Trusted Web Activity (TWA).
To store information about in-game progression, language, in-game virtual coins, use of an ad blocker and game version, we use cookies stored locally on your browser as the-mute-bow.com.
Game files are also stored in cache thanks to a service worker to allow the user to play offline.
No information about the user except the mentioned above is requested, processed or saved.
